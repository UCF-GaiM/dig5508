Complete the exercises from Chapter 2 using shapes in p5.js instead of strings. You can find it here:

https://eloquentjavascript.net/02_program_structure.html

There are hints that should be referenced if needed; I won't reproduce them here.

1-1: Looping a triangle

(to be placed in labs/1/1-1)

Write a loop that makes seven calls to a p5.js shape to output the following pattern (a right triangle): 

#
##
###
####
#####
######
#######

It may be useful to know that you can find the length of a string by writing .length after it. 

let abc = "abc";
console.log(abc.length);
// → 3

Most exercises contain a piece of code that you can modify to solve the exercise. Remember that you can click code blocks to edit them.

1-2: FizzBuzz

(to be placed in labs/1/1-2)

Write a program that uses the text function to print all the numbers from 1 to 100, with two exceptions. For numbers divisible by 3, draw a circle instead of the number, and for numbers divisible by 5 (and not 3), draw a square instead.

When you have that working, modify your program to print a triangle instead for numbers that are divisible by both 3 and 5 (and still print a circle or square for numbers divisible by only one of those).

(This is actually an interview question that has been claimed to weed out a significant percentage of programmer candidates. So if you solved it, your labor market value just went up.)

1-3: Chessboard

(to be placed in labs/1/1-3)

Write a program that creates a pattern of shapes that represents an 8×8 grid. At each position of the grid there is either a space or a shape . The shapes should form a chessboard. The following is one example of a textual version of this grid: 

 # # # #
# # # # 
 # # # #
# # # # 
 # # # #
# # # # 
 # # # #
# # # #

When you have a program that generates this pattern, define a binding size = 8 and change the program so that it works for any size, outputting a grid of the given width and height, and demonstrate it for two different sizes.

1-4: For the challenge problem:

(to be placed in labs/1/1-4)

Using loops, create a repeating pattern that changes based on the mouse position. You may use the mouse's location relative to the center, or the x and y components, but it must change every element in the repeating design (step and repeat are suggested).
